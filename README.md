React Skelton
===================

React, Flux, React-Router, HTML5 History API, Browserify, Watchify, Bower, Sass, Twitter Bootstrap, Font-Awesome, ES6, ESLint Example.

RUN
---------

```
$ npm install
$ npm run build
$ node server.js
```

Watch editing
---------

```
$ npm run watch
```
